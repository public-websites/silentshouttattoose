2. Vilka personuppgifter samlar vi in?

Vi samlar endast in de personuppgifter som du frivilligt lämnar till oss via vårt kontaktformulär:

  • Namn - för att vi ska kunna kontakta dig personligt
  • E-postadress - för att kunna svara på din förfrågan
  • Telefonnummer - för direktkontakt vid behov
  • Meddelande/förfrågan - för att förstå dina önskemål och behov

3. Varför behandlar vi dina personuppgifter?

Vi behandlar dina personuppgifter för följande ändamål:

  • Att svara på dina frågor och förfrågningar
  • Att boka tid för tatueringar eller konstuppdrag
  • Att kommunicera med dig om ditt projekt
  • Att följa upp din kundservice

Rättslig grund: Ditt samtycke enligt GDPR artikel 6.1(a) och vårt berättigade intresse att bedriva verksamhet enligt artikel 6.1(f).

4. Hur lagras dina personuppgifter?

Dina personuppgifter:

  • Skickas via säker e-post till silentshouttattoo@gmail.com
  • Lagras i Gmail:s säkra molntjänst
  • Hanteras endast av Kjell Saxberg
  • Sparas inte i några andra databaser eller system

5. Hur länge sparar vi dina uppgifter?

Vi sparar dina personuppgifter:

  • Aktiva kunder: Under pågående kundrelation plus 3 år för eventuell efterservice
  • Förfrågningar som inte leder till bokning: Upp till 1 år
  • Avslutade kundrelationer: Upp till 3 år för dokumentation och referens

6. Delar vi dina uppgifter med tredje part?

Nej, vi delar inte dina personuppgifter med tredje part. Undantag kan göras endast:

  • Om det krävs enligt lag
  • Vid myndighetsförfrågan
  • Med ditt uttryckliga samtycke

7. Dina rättigheter enligt GDPR

Du har rätt att:

  • Få tillgång till dina personuppgifter
  • Rätta felaktiga uppgifter
  • Radera dina uppgifter ("rätten att bli glömd")
  • Begränsa behandling av dina uppgifter
  • Invända mot behandling
  • Dataportabilitet - få ut dina uppgifter i strukturerat format
  • Återkalla samtycke när som helst

8. Säkerhet

Vi vidtar lämpliga tekniska och organisatoriska säkerhetsåtgärder för att skydda dina personuppgifter:

  • Säker e-postöverföring via EmailJS
  • Google:s säkerhetsstandarder för Gmail
  • Begränsad åtkomst - endast Kjell Saxberg har tillgång
  • Regelbunden säkerhetsöversyn

9. Cookies och spårning

Vår webbplats använder inga cookies för spårning eller analys. Vi samlar inte in någon information om ditt besök utan ditt medgivande.

10. Ändringar i integritetspolicyn

Vi kan komma att uppdatera denna integritetspolicy. Vid väsentliga ändringar kommer vi att informera dig via e-post eller genom tydlig information på webbplatsen.
